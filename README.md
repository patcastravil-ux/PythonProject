# Мой первый проект на Python: Калькулятор 🧮

Привет! Это мой учебный проект. Я учусь в 7 классе и осваиваю Python и Git.

## Что умеет этот калькулятор:
- Складывать, вычитать, умножать и делить.
- Работает в бесконечном цикле, пока не напишешь `exit`.
- Защищен от ошибок (если ввести буквы вместо чисел).

## Как запустить:
1. Установите Python.
2. Запустите файл `main.py`.
